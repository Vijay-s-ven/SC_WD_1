# Responsive Landing Page with Interactive Navigation

##  Project Overview
This project is a modern, responsive landing page featuring a fixed interactive navigation menu that dynamically responds to user actions such as scrolling and hovering, ensuring a smooth and engaging user experience across all devices.

##  Features
- Fixed navigation bar visible across all sections
- Dynamic navbar style change on scroll
- Active link highlighting based on scroll position
- Smooth scrolling between sections
- Responsive design with mobile hamburger menu
- Clean and modern UI with CSS animations

##  Technologies Used
- **HTML5** – Page structure and semantic layout  
- **CSS3** – Styling, responsiveness, transitions, and animations  
- **JavaScript (ES6)** – Scroll detection and interactive behavior
